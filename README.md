## Built With:

- Ruby on Rails w/ Bootstrap
- MySQL8 Database
- Rspec Test

## Prerequisites:
Please ensure that all requirements are satisfied  <br>
- rails >= 6.1.3.2
- mysql2 == 0.5
- puma == 5.0
- saas-rails >= 6
- webpacker == 5.0
- turbolinks == 5
- jbuilder == 2.7
- bcrypt == 3.1.7
- byebug
- rspec-rails
- rails-controller-testing
- web-console >= 4.1.0
- rack-mini-profiler == 2.0
- listen == 3.3
- spring
- spring-commands-rspec
- database_cleaner
- factory_bot_rails
- capybara >= 3.26
- selenium-webdriver
- webdrivers
- tzinfo-data, platforms: [:mingw, :mswin, :x64_mingw, :jruby]

## Build from sources
```bash
rbenv install 2.7.2
gem install bundler:2.2.19
bundle install
rails db:create
rails db:migrate
yarn install
rails s
```
